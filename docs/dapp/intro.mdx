<Alert severity='info'><strong>Contracts that work on EVM-compatible blockchains should work on Arthera with minimal or no changes.</strong></Alert>
<br/>

Arthera is an L1 blockchain that provides full EVM support. For developers, this means that building on Arthera is the same as building on Ethereum.
Contracts that work on Ethereum or other EVM-compatible blockchains should work on Arthera with minimal or no changes.

You can use your favorite Ethereum development tools to work with smart contracts on Arthera.
- [**Hardhat**](https://hardhat.org/)
- [**Truffle**](https://trufflesuite.com/)
- [**Alchemy**](https://docs.alchemy.com/reference/alchemy-sdk-quickstart)
- [**Remix**](https://remix.ethereum.org/)
- [**OpenZeppelin**](https://docs.openzeppelin.com/contracts/4.x/)
